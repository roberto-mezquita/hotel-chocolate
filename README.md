# Hotel Chocolate

Sistema de gestión hotelera desarrollado en C# utilizando Windows Forms.

## Descripción

Hotel Chocolate es una aplicación de escritorio diseñada para facilitar la administración de un hotel mediante la gestión de usuarios, reservas, pagos y reportes.

## Funcionalidades

* Gestión de usuarios y roles
* Registro y administración de reservas
* Control de pagos
* Generación de reportes
* Almacenamiento de información en base de datos

## Tecnologías utilizadas

* C#
* Windows Forms
* SQL Server
* Visual Studio
